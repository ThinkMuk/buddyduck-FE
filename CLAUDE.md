# Concert Buddy

콘서트 동행 매칭 앱 (콘서트에 관심있는 누구나, 20대 주 타겟)

## 폴더 구조
- 디자인 스펙 → design/CLAUDE.md
- 개발 컨벤션 → dev/CLAUDE.md

## 화면 범위
CB-01(로그인) ~ CB-14'(프로필 편집), 총 20개 화면

## 중요
- `dev/` 하위 구현/수정 작업 시 반드시 dev/CLAUDE.md의 개발 컨벤션을 함께 적용
- 디자인 변경 시 반드시 design/CLAUDE.md 업데이트
- 새 컴포넌트 만들면 dev/CLAUDE.md 목록에 추가
